### Descriptions (in progress)

- **Singleton Pattern**  
  인스턴스를 하나 생성한 뒤 이미 생성한 인스터스가 있을 경우 기존의 인스턴스를 사용한다.
- **Observer Pattern(Pub/Sub)**  
  이벤트 핸들러가 이벤트를 발생 시키고 이벤트에 대한 인수를 전달하여 Pub/Sub 형태로 작동한다. (e.g. 클릭 이벤트가 발생한 곳의 좌표)
